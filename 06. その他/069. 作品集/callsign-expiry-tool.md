# Callsign Expiry Tool ブラウザ版

by JA1XPM

## 概要

コールサインの有効期限切れを調べるため、総務省の無線局等情報検索で一度に照会するWebブラウザ用ツールです。
クラブ員の免許切れを防ぐ目的で使用します。

検索結果を保存する機能はありません。必要な結果はコピーして利用してください。

## 必要な環境

- Windows 10以降
- Google ChromeまたはMicrosoft Edge
- インターネット接続

ファイルはすべて同じフォルダに置いてください。

## 使い方

1. `index.html` をダブルクリックします。
2. 「CSVを選択」を押し、調べるCSVファイルを選びます。
3. 「検索開始」を押します。
4. 検索完了後、「結果テキスト」欄に結果が表示されます。
5. 必要な範囲を選択し、Ctrl+Cでコピーします。

## 入力CSV

ヘッダーを付けず、1列目に名前、2列目にコールサインを入れます。

```csv
JA1XPM,JA1XPM
横浜市役所クラブ,JR1YWC
```

UTF-8、UTF-8 BOM付き、Shift_JISのCSVを読み込めます。

## 検索結果

結果テキストには検索年月日、名前、コールサイン、有効期限、残り期間、局種別が表示されます。
同じコールサインで複数の局が見つかった場合は局ごとに表示し、見つからない場合は「有効期限切れ」と表示します。

結果の末尾には次の再掲欄を表示します。

- 有効期限まで6か月を切った局
- 有効期限切れの局

## 検索間隔

総務省サイトへは1件ずつアクセスします。検索間隔の初期値は3秒から5秒です。
3秒未満は指定できません。短時間に大量の検索を行わず、1日2回程度の使用にとどめてください。
