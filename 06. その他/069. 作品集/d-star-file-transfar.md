# DSX DV Fast File Transfer Ver.1.0

## 概要

D-STAR DV Fastの簡易データ通信で使う、送信専用CLIと受信専用CLIです。
X/YMODEMではなく、先頭で短い速度確認を行ったあと、ファイル本体をburst送信し、最後に欠落したパケットだけ再送します。

## 事前準備

- 送信側・受信側ともに、リグをD-STAR簡易データ通信ができる設定にします。
- USB等でWindowsパソコンとリグを接続します。
- プログラムを起動します。

## ファイル

- `dsx_send.exe`: 送信側
- `dsx_recv.exe`: 受信側

## 起動方法

受信側を先に起動します。

```text
dsx_recv.exe COM10 9600
```

次に送信側を起動します。

```text
dsx_send.exe COM11 9600 test.png
```

## 注意

- 受信側のファイル名は送信側から送られます。
- 同名ファイルがある場合は上書きせず、`.recv1` などを付けて保存します。
- 受信完了時にSHA-256で元ファイルとの完全一致を確認します。
- D-STARの仕様上、データ通信に使用できない `0x7E`、`0x7D`、`0x11`、`0x13` は通信中にエスケープします。
- 現在は送信ファイルを圧縮しません。必要に応じて任意の圧縮ソフトで圧縮してください。

## プロトコル

1. 送信側がファイル名、サイズ、SHA-256を送ります。
2. 受信側が応答します。
3. 送信側が短いprobe通信で速度を確認します。
4. ファイル本体をburst送信します。
5. 受信側が欠落パケット番号だけ返します。
6. 送信側が欠落分だけ再送します。
7. SHA-256の一致を確認して完了します。
