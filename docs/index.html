協力会について

横浜市アマチュア無線非常通信協力会は、災害時における通信支援を目的として活動するボランティア団体です。アマチュア無線の技術と情熱を活かし、地域の安全と安心に貢献しています。

活動内容

  • 災害時の非常通信支援
  • 定期的な訓練・演習の実施
  • 地域防災イベントへの参加
  • 技術講習会・ライセンス取得支援

入会案内

アマチュア無線に興味があり、地域の防災に貢献したい方はぜひご参加ください。ライセンスの有無に関わらず、意欲ある方を歓迎します。

GitHub
